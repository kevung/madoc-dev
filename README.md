# Mkdocs préconfigurée

## Résumé

Ce dépot contient une documentation Mkdocs vierge préconfigurée (thème, latex, footnote...)

## Pré-requis

- python (3.3+)
- [pip](https://pypi.org/)

## Installation

```
pip install -r requirements.txt
```

## Visualisation de la doc

```
python -m mkdocs serve
```
